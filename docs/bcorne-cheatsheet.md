# BCORNE — шпаргалка раскладки

Печатаешь = базовый слой. Символы/F = держишь **MO2** (правый палец).
Alt = держишь **Fn** (левый палец). Язык = **Esc+Space**.

---

## СЛОЙ 1 — База, English
```
 1   2   3   4   5         6   7   8   9   0
 q   w   e   r   t         y   u   i   o   p   [⏎]   ← правый Enter = [
 a   s   d   f   g         h   j   k   l   ;   '
 z   x   c   v   b         n   m   ,   .   /
        ┌ Ctrl Alt Space   Enter MO2 Super ┐
```

## СЛОЙ 1 — База, Русский (та же клавиша)
```
 1   2   3   4   5         6   7   8   9   0
 й   ц   у   к   е         н   г   ш   щ   з   [х]   ← правый Enter = х
 ф   ы   в   а   п         р   о   л   д   ж   э
 я   ч   с   м   и         т   ь   б   ю   .
```
Спец-буквы:
- **х** = правый **Enter** (см. схему выше)
- **ъ** = **MO2 + K**
- **ё** = **MO2 + A**

---

## СЛОЙ 2 — MO2 (символы + F). Одинаково в EN и RU.
```
держишь MO2 +
 цифры:  1   2   3   4   5      6   7   8   9   0
   →     F1  F2  F3  F4  F5     F6  F7  F8  F9  F10

 буквы:  !   @   #   $   %      ^   &   *   (   )      (ряд qwert / yuiop)
         `   \   /   =   +      -   [   ]   {   }      (ряд asdfg / hjkl;)
         ~   |   _   <   >      :   '   "   ;   ?      (ряд zxcvb / nm,./)
```
Часто нужное:
`-`=MO2+H  `_`=MO2+C  `=`=MO2+F  `+`=MO2+G
`[`=MO2+J  `]`=MO2+K  `(`=MO2+O  `)`=MO2+P
`{`=MO2+L  `}`=MO2+;  `\`=MO2+S  `/`=MO2+D

---

## Fn (левый палец) = Alt — модификатор, своих букв нет
Держишь Fn + клавиша = **Alt+клавиша** (работает в EN и RU, т.к. по кейкоду):
- Fn+C → Alt+C (fzf)
- Fn+D / Fn+V / Fn+P → rofi / буфер / power
- Fn+B / Fn+F → Alt по словам ←/→
- Fn+Стрелки → Alt+Стрелки

---

## Переключение языка
- **Esc + Space** вместе → RU ⇄ EN
- RGB: **холодный белый = EN**, **тёплый янтарь = RU**

## Ручки / прочее
- **Клик левой ручки** = mute; крутить = громкость
- **Двойной тап правого Super** → workspace "terminal"
